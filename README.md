# 行道难

RIME 输入方案

**该方案仅包含 8040 个汉字,汉字来源为 「现代汉字词频表」、国标汉字**

注：已开启用户字典，初期打字较慢但随用户造词提升速度

## 一、拼‧行道难

全拼方案，与日常使用的拼音输入法使用方法一致

## 二、双‧行道难

双拼方案，与「国标双拼」一致，附国标双拼健位图。

![国标双拼健位图](https://raw.githubusercontent.com/Git-So/way_pinyin/main/assets/double_pinyin.png)

注：零声母使用 `A + [对应键]`，如：

```
a       AA
o       AO
e       AE
ai      AK
er      AL
ang     AG
```
